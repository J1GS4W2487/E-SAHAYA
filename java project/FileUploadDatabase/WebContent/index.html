E-Sahay — How it works!

Reporting Grievances in your area has never been so easy! No need for formal letters and long wait times. Now be an active citizen with minmal effort, using these four steps given below!

Step 1:

Click on Create grievance on the home page.

Step 2:

Login. If you do not have an account you can create one using the register now link

Step 3:

Fill the given form and upload suitable images. Make sure that the images are clear and recognisable

Step 4:

Click on submit and be proud of yourself for being an active citizen!
